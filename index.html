How to Use This Tool

  • Select a file: Choose a .txt or .csv file from your computer
  • For CSV files: Select the column you want to analyze
  • Hyphenated words: Words like "well-known" are treated as single words
  • Generate unigrams: Click the "Generate Unigrams" button to analyze word frequencies and TF-IDF
  • Generate bigrams: Click the "Generate Bigrams" button to analyze word pairs
  • Generate trigrams: Click the "Generate Trigrams" button to analyze word triplets
  • Generate 4-grams: Click the "Generate 4-grams" button to analyze word quadruplets
  • Click on any N-gram: Click on a bigram, trigram, or 4-gram to see it in context (KWIC)
  • KWIC Analysis: Enter a word or phrase with wildcards (*) and click "KWIC Analysis"
  • Wildcard examples: polit* matches politics, political; *ec* matches selected, projected; eq* matches equal, equity
  • Adjust display: Set the number of items to show in the table
  • Filter results: Set minimum frequency and score values, then click "Refresh"
  • Sort Click column headers to sort by that column (click again to reverse order)
